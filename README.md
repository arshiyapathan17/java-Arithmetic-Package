# java-Arithmetic-Package
Java project demonstrating a user defined package to perform basic arithmetic operations. Created for the Advanced Java Weekly Commit Challenge (Week 3).
Java project demonstrating the use of a user defined package to perform basic arithmetic operations.

Subject: Advanced Java  
Class: BSc ECS-III (SEM-VI)  
Week: 3 – GitHub Weekly Commit Challenge  

## Features
- User defined package `mycalculator`
- Arithmetic operations:
  - Addition
  - Subtraction
  - Multiplication
  - Division
  - Modulus

## Project Structure
Week3-Arithmetic-Package-Java
│
├── MainApp.java
